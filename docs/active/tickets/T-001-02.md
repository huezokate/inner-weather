## Context

Build Plan Task 2. Add real Reddit posts to the feed using the public JSON endpoint (no key,
always runs). Calm subs should slot under the ceiling; hot-take subs get shielded in Fog —
which exercises the shield with real content.

## Implementation notes

- New file `src/sources/reddit.ts`. For each `REDDIT_LANES` entry in `feed.ts`: GET
  `https://www.reddit.com/r/{sub}/hot.json?limit=4`, map posts → `FeedItem`
  (`title`, `url`, thumbnail if present, `intensity` = the lane's prior, `source: "reddit"`,
  `kind` = lane kind). Wrap each fetch in try/catch; a failing lane returns `[]`.
- Wire into `fetchLiveFeed()` in `feed.ts` (replace the placeholder `return []`). Dedupe by
  url/title; cap the live set at ~20 items. Live items append to HERO_FEED, never replace it.
- ⚠️ CORS: `www.reddit.com/*.json` is usually browser-fetchable; if blocked, add a Vite dev
  proxy in `vite.config.ts` like the Oura one.

## Acceptance Criteria

- [ ] `src/sources/reddit.ts` exists; `fetchLiveFeed()` returns real Reddit posts.
- [ ] Posts appear alongside the hero set; calm subs sit under the ceiling, hot-take subs are
      shielded in Fog.
- [ ] Live feed is deduped and capped (~20).
- [ ] A failed lane/fetch falls back cleanly; HERO_FEED + flip still work.
- [ ] `npx tsc -b` is clean and the dev server boots.
